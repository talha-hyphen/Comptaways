# Comptaways

A simple website project for Comptaways.

## Project Structure

- index.html - Home page
- about.html - About page
- services.html - Services page
- contact.html - Contact page
- assests/ - CSS, JS, images, and other assets

## Getting Started

Open the project folder in your browser or serve it using a local web server.
